# Space Asteroids

A simple game made in python3 with [Ursina](https://www.ursinaengine.org/).

__Download:__ To try it yourself, simply: 
- Go to the [releases](https://github.com/Alex-M-2013/) section 
- Download and extract the ZIP file
- Look for 'space asteroids.exe' inside the extracted folder and run it

## __Features include:__

- Asteroids constantly spawning that you have to dodge
- A main menu and death screen
- A score counter
- Screen wrapping
- Self-made assets

__Note:__ This is my first game and I am not the best programmer so the code is quite messy 😅 (But still works!).
 The collisions are also a little bugged.
